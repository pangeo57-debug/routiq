# Πολιτική Απορρήτου — RoutePal

**Τελευταία ενημέρωση**: [ΣΥΜΠΛΗΡΩΣΕ ΗΜΕΡΟΜΗΝΙΑ]

## 1. Ποιοι είμαστε

Το RoutePal ("εμείς", "η Εφαρμογή") είναι μια εφαρμογή προγραμματισμού και βελτιστοποίησης δρομολογίων για επαγγελματίες (εκπαιδευτικούς, φυσιοθεραπευτές, και άλλους ελεύθερους επαγγελματίες — "Χρήστες").

**Υπεύθυνος επεξεργασίας δεδομένων**: [ΟΝΟΜΑ / ΕΠΩΝΥΜΙΑ ΕΠΙΧΕΙΡΗΣΗΣ], [ΔΙΕΥΘΥΝΣΗ], [ΑΦΜ αν υπάρχει]
**Στοιχεία επικοινωνίας για θέματα προστασίας δεδομένων**: [EMAIL]

## 2. Τι δεδομένα συλλέγουμε

### 2.1 Δεδομένα του Χρήστη (του επαγγελματία)
- Όνομα, διεύθυνση email, διεύθυνση κατοικίας/εργασίας
- Ρυθμίσεις εφαρμογής (ωράριο εργασίας, προτιμήσεις)
- Στοιχεία πληρωμής (αν υπάρχει συνδρομή) — επεξεργάζονται από τρίτο πάροχο πληρωμών (π.χ. Stripe), δεν αποθηκεύουμε εμείς στοιχεία κάρτας

### 2.2 Δεδομένα Τρίτων (μαθητών/πελατών του Χρήστη)
**Σημαντικό**: ο Χρήστης εισάγει στην εφαρμογή δεδομένα **τρίτων προσώπων** — των δικών του μαθητών/πελατών — που **ενδέχεται να είναι ανήλικοι**. Αυτά περιλαμβάνουν:
- Όνομα, διεύθυνση
- Στοιχεία μαθήματος/ραντεβού (μάθημα, διάρκεια, συχνότητα, διαθεσιμότητα)
- Προαιρετικές σημειώσεις

**Ο Χρήστης είναι υπεύθυνος να έχει τη νόμιμη βάση** (π.χ. συγκατάθεση γονέα/κηδεμόνα για ανήλικους) για να εισάγει αυτά τα δεδομένα στην εφαρμογή. Εμείς λειτουργούμε ως **εκτελούντες την επεξεργασία** (data processor) για αυτά τα δεδομένα, όχι ως υπεύθυνοι επεξεργασίας.

## 3. Πώς χρησιμοποιούμε τα δεδομένα

- Για να παρέχουμε τη λειτουργία προγραμματισμού/δρομολόγησης
- Για γεωκωδικοποίηση διευθύνσεων (μετατροπή διεύθυνσης σε συντεταγμένες χάρτη) μέσω τρίτων παρόχων (βλ. §4)
- Για τη λειτουργία λογαριασμού/σύνδεσης (αν υπάρχει)
- Για επικοινωνία σχετική με την υπηρεσία (ενημερώσεις, τεχνικά θέματα)

**Δεν** χρησιμοποιούμε τα δεδομένα για διαφήμιση ή πωλούμε δεδομένα σε τρίτους.

## 4. Τρίτοι πάροχοι (Data Processors)

| Πάροχος | Σκοπός | Δεδομένα που λαμβάνει |
|---|---|---|
| OpenStreetMap / Nominatim | Γεωκωδικοποίηση διευθύνσεων | Κείμενο διεύθυνσης |
| HERE Technologies (προαιρετικό) | Γεωκωδικοποίηση, δρομολόγηση με live traffic | Κείμενο διεύθυνσης, συντεταγμένες |
| [Stripe — όταν προστεθεί] | Επεξεργασία πληρωμών | Στοιχεία πληρωμής Χρήστη (όχι μαθητών) |
| [Πάροχος hosting/backend — όταν προστεθεί] | Φιλοξενία server/βάσης δεδομένων | Όλα τα παραπάνω δεδομένα |

*[ΣΗΜΕΙΩΣΗ ΓΙΑ ΤΟΝ ΔΙΚΗΓΟΡΟ: χρειάζεται Data Processing Agreement (DPA) με κάθε πάροχο που επεξεργάζεται προσωπικά δεδομένα για λογαριασμό μας.]*

## 5. Πού αποθηκεύονται τα δεδομένα

*[ΤΡΕΧΟΥΣΑ ΚΑΤΑΣΤΑΣΗ: η εφαρμογή σήμερα αποθηκεύει δεδομένα τοπικά στη συσκευή του Χρήστη (localStorage), όχι σε κεντρικό server. Αυτή η ενότητα πρέπει να ενημερωθεί όταν προστεθεί backend/βάση δεδομένων, συμπεριλαμβανομένου του πού γεωγραφικά βρίσκεται ο server (εντός ΕΕ προτιμητέο για απλούστερη συμμόρφωση GDPR).]*

## 6. Πόσο καιρό κρατάμε τα δεδομένα

*[ΣΥΜΠΛΗΡΩΣΕ: π.χ. "Όσο ο λογαριασμός παραμένει ενεργός, και για Χ μήνες μετά τη διαγραφή λογαριασμού για λόγους... "]*

## 7. Δικαιώματα των υποκειμένων δεδομένων

Κάθε φυσικό πρόσωπο του οποίου τα δεδομένα επεξεργαζόμαστε έχει δικαίωμα:
- **Πρόσβασης** — να μάθει τι δεδομένα κρατάμε γι' αυτόν
- **Διόρθωσης** — να διορθώσει λανθασμένα δεδομένα
- **Διαγραφής** ("δικαίωμα στη λήθη") — να ζητήσει διαγραφή
- **Φορητότητας** — να λάβει τα δεδομένα του σε αναγνώσιμη μορφή
- **Εναντίωσης** στην επεξεργασία

Για αιτήματα σχετικά με δεδομένα **μαθητή/τρίτου**, το αίτημα πρέπει να απευθύνεται στον Χρήστη (τον υπεύθυνο επαγγελματία) που εισήγαγε τα δεδομένα, ή σε εμάς στο [EMAIL] — θα το προωθήσουμε στον αρμόδιο Χρήστη.
